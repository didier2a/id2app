Complémentarité

Un contrôle complémentaire, pas une opposition à votre infogérant

INFOSERV2A ne cherche pas à remplacer l’administration informatique existante. L’approche est complémentaire : vérifier localement les points que les outils distants, les tableaux de bord et les procédures générales ne couvrent pas toujours.

Cette posture facilite le dialogue. Les constats peuvent être transmis au prestataire informatique pour prioriser les corrections, documenter les risques et améliorer le niveau de sécurité sans créer de conflit inutile.

Contrôle terrain

Ce que nous vérifions

Le risque n’est pas uniquement dans le cloud ou les serveurs. Il existe aussi dans les locaux, les usages et les équipements oubliés.

  • inventaire réseau
  • équipements inconnus
  • ports et services ouverts
  • WiFi invité et WiFi interne
  • accès distants
  • comptes et MFA
  • sauvegardes
  • imprimantes, caméras, NAS
  • messagerie
  • dépendance prestataire
  • documentation minimale
